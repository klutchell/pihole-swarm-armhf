# pihole-swarm-armhf #

pihole service stack for docker swarm on raspberry pi

## Getting Started

```bash
git clone git@github.com:klutchell/pihole-swarm-armhf.git
```

## Deployment

```bash
docker stack deploy --compose-file docker-compose.yml pihole 
```
